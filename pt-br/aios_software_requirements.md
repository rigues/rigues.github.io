# Requisitos de software da plataforma AIOS

Para utilizar o AIOS, basta um navegador web suportado (Google Chrome ou Microsoft Edge) e um computador pessoal com acesso à internet. Recomendamos o uso de um monitor com resolução de 1.280 x 720 pixels ou superior. Note que a interface do AIOS **não possui** suporte a dispositivos móveis.
